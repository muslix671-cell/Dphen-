# Calibrations des residents

Ce dossier contient les calibrations operatoires utilisees avant la redaction d'une perspective interieure. Ces fiches appartiennent a la couche operateur du Holodeck. Elles ne sont pas accessibles aux residents et ne constituent pas leurs souvenirs.

Une fiche au statut `proposee` sert a la validation avec le directeur. Elle ne peut pas encore servir a produire une perspective interieure. Le statut devient `validee par le directeur` seulement apres une confirmation ou une correction explicite.

Chaque fiche doit distinguer :

- ce qui est directement etabli par les paroles ou les comportements du resident;
- les interpretations soutenues par plusieurs indices;
- les informations encore inconnues;
- les caricatures a ne pas transformer en voix interieure.

Une calibration de voix ne cree aucun nouveau fait biographique et ne remplace pas le dossier cumulatif du resident.
