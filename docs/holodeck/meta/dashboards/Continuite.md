## Vérification avant une scène

Liste de référence à vérifier avant chaque scène :

- Le bon programme est chargé.
- Le Runtime et le contexte actif ont été consultés.
- Les updates en attente ont été vérifiés.
- Seules les séances actives pertinentes ont été retenues.
- Chaque membre reçoit uniquement les informations auxquelles il a accès.
- Aucune archive technique ou remplacée n’est chargée comme continuité.
- Aucune information meta n’entre dans la scène.
