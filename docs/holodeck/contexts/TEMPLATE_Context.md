# Context Template

This file defines a Holodeck program context.

## Program name

Name of the program.

## Purpose

What this program is for.

## Sources

Project sources or documents that should be consulted.

## Persistent members

Recurring workshop members for this program, if any.

For each member, record the path to the resident file created from `docs/holodeck/residents/TEMPLATE_Resident.md`.

## Resident location

Path where resident files owned by this program are stored.

## Invited specialists

Specialists commonly useful for this program.

## Context-specific rules

Rules that apply only to this program.

## Journal location

Path to the journal for this program.

## Memories location

Path to the memories for this program.

## Session location

Path to session records for this program.

## Update naming convention

Expected naming format for pending updates.

## Anti-contamination rules

What must not be imported from other programs unless explicitly requested.
