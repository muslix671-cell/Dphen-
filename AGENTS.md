# Instructions for Codex

This repository contains the DPhen tabletop RPG project.

## Holodeck workflow

Codex may modify files under:

- docs/holodeck/

Codex must not modify official DPhen rules, lore, or source documents unless explicitly instructed.

When applying a Holodeck update:

1. Read the update file in docs/holodeck/pending_updates/.
2. Update only the relevant Holodeck files.
3. Preserve existing structure and tone.
4. Do not invent decisions, memories, or character development.
5. If information is ambiguous, leave a note in Journal.md under "À clarifier".
6. Show a diff before merge.
