# Instructions for Codex

This repository contains the DPhen tabletop RPG project.

## Holodeck workflow

Codex may modify files under `docs/holodeck/` only when the user explicitly asks Codex to apply a Holodeck update.

Codex must not modify official DPhen rules, lore, source documents, or `.docx` files unless the user gives a separate explicit instruction.

When applying a Holodeck update:

1. Read the requested update or pending note first.
2. Update only the relevant files under `docs/holodeck/`.
3. Preserve the existing structure and tone.
4. Do not invent decisions, memories, or character development.
5. If information is ambiguous, leave a note in `docs/holodeck/Journal.md` under `A clarifier`.
6. Show a diff before commit.
